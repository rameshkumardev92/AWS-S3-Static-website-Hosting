# AWS S3 Static Website Hosting and Storage Management

## Project Overview 
This Project Demonstrates hosting a static websiste using AWS S3 with proper IAM configuration and monitoring using CloudWatch.

## Tools used 
-- AWS S3
-- AWS IAM
-- AWS CloudWatch

##  Project Features
 -- Created S3 bucket and enabled static website hosting 
 -- Configured bucket policy for public access
 -- Implemented IAM roles and policies for secured access
 -- Upload HTML fies (index.html)
 -- Enabled logging and monitoring using Cloud watch
 -- Managed Object storage and permission
